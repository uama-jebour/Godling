# 在家工作设置指南

## 首次在家工作设置步骤

### 1. 同步代码

```bash
cd /path/to/Godling
git pull origin main
```

### 2. 安装 Azure ImageGen Skill

```bash
bash scripts/tools/install_azure_imagegen_skill.sh
```

### 3. 设置环境变量

编辑 `~/.zshrc`，添加：

```bash
# Azure OpenAI Image Generation
export AZURE_OPENAI_API_KEY="你的API密钥"
export AZURE_OPENAI_ENDPOINT="https://你的资源名.openai.azure.com"
export AZURE_OPENAI_API_VERSION="2025-04-01-preview"
export AZURE_OPENAI_IMAGE_DEPLOYMENT="gpt-image-1.5"
```

然后执行：

```bash
source ~/.zshrc
```

### 4. 验证环境

```bash
# 测试 Godot
/Applications/Godot.app/Contents/MacOS/Godot --version

# 测试 Azure skill dry-run
python3 ~/.codex/skills/azure-imagegen/scripts/azure_image_gen.py generate \
  --prompt "test" --out /tmp/test.png --dry-run

# 运行冒烟测试
/Applications/Godot.app/Contents/MacOS/Godot --headless --path . \
  --script tests/bootstrap_scene_smoke_test.gd
```

## 故障排除

### Godot 命令找不到

确保使用完整路径：
```bash
/Applications/Godot.app/Contents/MacOS/Godot
```

### Azure skill 报错

1. 检查环境变量是否设置：`echo $AZURE_OPENAI_API_KEY`
2. 重新安装 skill：`bash scripts/tools/install_azure_imagegen_skill.sh`
3. 检查 Python openai 包：`pip3 install openai`

### 测试失败

1. 检查 Godot 版本是否为 4.6
2. 删除 `.godot/` 缓存目录重试
3. 查看具体错误输出
