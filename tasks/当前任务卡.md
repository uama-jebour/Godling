# 当前任务卡

## 当前阶段目标

当前默认优先任务是：

`巩固最小可玩闭环，并补齐地图首屏 / 交互战斗 / Web 发布链路`

## Goal

完成“首屏地图可选事件 + 进入交互战斗 + 结果回写 + 可导出 Web 并准备发布”的稳定原型。

## Scope

只做：

- 首屏地图事件板与右侧事件面板联动
- 交互战斗的基础操作、结算与回写
- 首屏布局与窗口适配
- Web 导出预设与 GitHub Pages 发布链路
- 文档与交接同步

不做：

- 完整职业/技能树
- 完整背包拖拽 UI
- 复杂 AI 与高级战斗特效
- 最终美术资源替换
- 完整剧情分支演出

## 验收标准

- 项目启动后首屏能显示地图事件板
- hover 地图事件点时，右侧事件面板实时联动
- 点击战斗事件后可进入交互战斗并正常回写结果
- 普通流程、强制袭击、撤离流程都保持可用
- 本地 `Web` 导出命令可成功产出浏览器构建
- GitHub Pages 工作流文件齐全，具备自动发布条件
