# 玩家流程样例

## 标准体验链路（A世界第2张地图）

该流程是当前文档和样例数据的权威体验参考。

1. 玩家进入家园界面，检查信息并配置英雄与携带物品，然后选择进入 `A世界` 的第 `2` 张地图。
2. 第一回合进入 2D 地区地图后，出现 `5` 个事件：`4` 个随机事件（`3` 战斗 + `1` 随机）和 `1` 个固定主线叙事事件。
3. 玩家选择一个战斗事件，弹出出击面板，查看风险与收益后出击。
4. 战斗结束后进入第二回合，地图刷新 `6` 个事件，其中主线叙事事件保留不变，其余事件重刷。
5. 到第四回合后，恶魔可能出现并触发额外强制战斗事件，该事件不占用回合选择次数。
6. 到第六回合，玩家获得主线提交物品并提交，然后点击主线任务进入主线战斗。
7. 主线完成后，玩家点击撤离并触发撤离事件（可为战斗突围）。
8. 撤离事件成功后，玩家带着背包中的物品回到家园并完成结算。

## 落地规则

- 固定主线事件默认跨回合保留，不被随机刷新覆盖。
- 强制事件属于 `forced_non_turn`，触发时为额外处理，不推进回合计数。
- 撤离不再是纯按钮成功，撤离事件可要求额外完成一次事件或战斗。
