# 游戏愿景

## 核心体验

玩家要感受到的不是“打一场自动战斗”，而是“在一张逐步变危险的地图里，围绕明确目标做一连串取舍，并把部分成果带出地图”。

这份体验由 4 根支柱组成：

1. 任务给目标
2. 事件给变化
3. 战斗给风险
4. 撤离给决策重量

## 灵感落点

- 来自《苏丹的游戏》的部分：任务包装叙事、通过推进逐渐理解世界和势力
- 来自类《鸭科夫》的部分：局内收益与撤离风险并存，继续深入始终有代价

## 玩家在单局中应感受到什么

- 起初是“先拿点稳妥收益”
- 中段是“要不要为了更好的奖励继续压一回合”
- 后段是“手里已经有值钱东西，但再打一回合也许能开出更大的故事和收益”

## 什么算做偏了

以下情况都说明实现方向偏了：

- 任务只是一个换皮按钮，没有叙事推进价值
- 叙事只是文本弹窗，没有影响事件池、地图数值或解锁
- 战斗很好玩，但撤离决策不重要
- 玩家每回合只是被动点一个最优事件，没有真正的风险判断
