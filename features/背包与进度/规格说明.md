# 背包与进度规格说明

## 1. 目标

该模块负责明确：

- 本局拿到的东西先放哪
- 成功撤离后哪些东西能留下
- 死亡后哪些东西会失去
- 装备和解锁如何跨局保留
- 家园阶段配置如何影响本局（英雄、携带物、装备）

## 2. 资源分层

### 局内临时层

- 临时物资
- 临时货币
- 临时藏品

写入位置：

- `RunState`

### 局外永久层

- 永久仓库
- 永久货币
- 永久藏品
- 已完成任务
- `story_flags`
- `unlock_flags`

写入位置：

- `ProgressionState`

## 3. 撤离与死亡

### 撤离成功

- 临时资源转入永久层
- 本局任务完成和解锁写入永久状态

### 死亡失败

- 临时资源清空
- `story_flags` 与 `unlock_flags` 默认保留

## 4. 装备

- 只有 `type = 10` 的藏品可装备
- 装备来源是永久仓库
- 开局时生成装备快照
- 战斗事件读取装备修正

## 5. 家园与撤离

- 家园界面负责设置本局英雄和携带物品。
- 地图中完成主线后点击撤离，应触发撤离事件而非直接成功。
- 撤离事件成功后，才将临时战利品转为永久存档。

## 6. 当前不做

- 背包拼图
- 复杂负荷系统
- 家园换装和仓库联动
