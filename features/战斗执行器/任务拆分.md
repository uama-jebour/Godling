# 战斗执行器任务拆分

## Task 1：战斗输入输出骨架

- 建立 `BattleRequest`
- 建立 `BattleResult`
- 建立世界层与战斗层交互入口

验收：

- 世界层能拉起战斗壳子并拿到结果对象

## Task 2：基础单位与移动

- 实现敌人、友方、英雄
- 实现不可重叠
- 实现步行和飞行

验收：

- 单位可生成、移动、占位

## Task 3：基础攻击与胜负

- 实现近战和平射
- 实现索敌和追杀
- 实现死亡与战斗结束

验收：

- 战斗能自然分出胜负

## Task 4：战斗内事件

- 接入召唤和 buff 事件
- 支持通知、响应、结束

验收：

- 战斗内事件能稳定触发并影响结果
