# 世界地图与事件板任务拆分

## Task 1：地图锚点与事件板状态

- 建立随机事件锚点结构
- 建立固定事件容器
- 建立 `TurnBoardState`

验收：

- 地图可同时容纳随机事件和固定事件

## Task 2：随机事件生成器

- 从随机池抽取 `4-6` 个事件
- 放入地图固定位置
- 支持基础权重

验收：

- 每回合随机事件数量正确

## Task 3：固定事件保留逻辑

- 根据进入条件生成固定事件
- 支持跨回合保留
- 支持完成后消失或替换

验收：

- 固定事件不会被随机刷新覆盖

## Task 4：事件选择与回合推进

- 每回合只能选择 `1` 个事件
- 事件处理后进入下一回合

验收：

- 事件板能稳定连续运行多个回合
